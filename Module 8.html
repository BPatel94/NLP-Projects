Exercise 3 The first part of the passage is the key sentence: You can imagine Piglet's joy when at last the ship came in sight of him.

This contains: Main clause Subordinate clause starting with when So we replace each clause with S.

Result (Compressed Form): S when S Meaning: Main clause - S Subordinate clause - S Structure for compressed sentence: S /
S SBAR /
when S

Explanation: S = main sentence SBAR = subordinate clause when S = time clause

Main Syntactic Constructions Used: The long sentence is built using these structures: Subordination: clauses introduced by when Coordination: using conjunctions like and Relative clauses: e.g., which was really part of the story Prepositional phrases (PP): in sight of him Participial phrases: slipping slowly out of the window

Exercise 4 The word "buffalo" plays three roles simultaneously: it's a proper noun (Buffalo, NY), a common noun (bison), and a verb (to bully/intimidate). Normalizing to lowercase, the grammar is: S - NP VP NP - N | NP RC (noun phrase, optionally modified by a relative clause) RC - NP V (relative clause: "that NP buffalo/intimidate") VP - V NP (verb phrase) N - "buffalo" (common noun: bison) PN - "buffalo" (proper noun: city name, used as adjective/modifier) NP - PN N (city + animal) V - "buffalo" (verb: to intimidate)
